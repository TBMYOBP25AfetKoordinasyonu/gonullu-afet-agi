# Gönüllü Afet koordinasyon Ağı
Platformda yer alan tüm sayfalar ve işlevleri:

1. Ana Sayfa
Temel olarak yardım iste ve gönüllü ol olarak iki seçenek sunacakç 
Üst kısımda Giriş yapma veya kayıt olma kısmı de olacak.

3. Yardım İsteyen Sayfası
Afetzedeler bu sayfadan yardım talebinde bulunur.

Konum bilgisi otomatik alınır. Konum otomatik alınamazsa:

İl, ilçe, mahalle ve diğer detaylar manuel olarak seçilebilir.

3. Yardım Talebi Sonrası Sayfa
Yardım talebinin başarıyla alındığını bildirir.

Kullanıcıya süreç hakkında bilgi verir.

4. Gönüllü Dashboard Sayfası
Gönüllüler, harita üzerinden kendilerine yakın konumda olan afetzedeleri görüntüler.

5. Koordinatör Sayfası (Admin Paneldir)
Koordinatörler, harita üzerinden tüm afet bölgelerini, yardım isteyem varsa onları ve yakınlardaki gönüllüleri görebilir.

6. Kayıt Olma Sayfası
Kullanıcı türüne göre (afetzede, gönüllü, koordinatör) yeni üyelik oluşturulmasını sağlar.

7. Giriş Yapma Sayfası
Kayıtlı kullanıcıların sisteme erişimini sağlar.
